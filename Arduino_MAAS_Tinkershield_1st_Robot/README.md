# Arduino MAAS Tinkershield 1st Robot Boilerplate

## Core Components
1. MAAS Tinkershield
2. Arduino Uno
3. [I2C LCD Screen]()
4. 9g Micro Servo
5. Servo Clamp Gripper
6. Dupont to Aligator Clip Cables
7. 9v Battery
8. 9v Battery Snap
